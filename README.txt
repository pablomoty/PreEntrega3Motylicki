Este es un proyecto de manejo de tareas, que permite agregar tareas a una lista, limpiar la lista de tareas existentes y mostrar las tareas almacenadas utilizando el local storage.

Características del proyecto:

-Se pueden agregar tareas con un nombre, una descripción y una categoría.
-Si algún campo del formulario de agregar tareas está vacío, se mostrará un mensaje de error.
-Las tareas agregadas se almacenarán en el local storage para que se puedan acceder más tarde.
-Las tareas almacenadas se mostrarán en la lista de tareas.
-Las tareas almacenadas se pueden eliminar de la lista haciendo clic en el botón "Limpiar tareas".
